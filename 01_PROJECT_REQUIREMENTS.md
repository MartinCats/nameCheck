# Name Checked - Product Requirements

## Vision
ระบบเช็คชื่อนักเรียนสำหรับครูประจำชั้น ใช้งานง่ายบน iPhone, iPad และ Desktop

## Core Features
### Authentication
- Login ด้วย Google
- Logout
- Session Management

### Classroom Management
- สร้างห้องเรียน
- แก้ไขห้องเรียน
- ลบห้องเรียน
- ครูสูงสุด 2 คนต่อห้อง

### Student Management
- เพิ่มนักเรียน
- เพิ่มหลายคนพร้อมกัน
- แก้ไขข้อมูล
- ปิดใช้งานนักเรียน

### Attendance
- มาเรียน
- มาสาย
- ลา
- ขาดเรียน
- แก้ไขข้อมูลย้อนหลัง
- เลือกวันที่

### Reports
- สรุปรายบุคคล
- สรุปรายวัน
- Attendance Log
- Excel Export
- PDF Export

### PWA
- Install บน iPhone
- Offline shell
- Mobile first
